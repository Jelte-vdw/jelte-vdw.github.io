Binair stelsel

Om te kunnen werken, moeten computers informatie kunnen doorsturen van het ene component naar het andere. Dit doen ze alleen niet door middel van letters of nummers zoals wij dat gewend zijn, maar door middel van hun eigen taal: binaire code.
Binaire code bestaat uit nullen en enen, ook wel bits genoemd. Het gebruikt een string van deze bits in specifieke volgordes om informatie te verwerken. Zo’n string van bits wordt een byte genoemd.
Een voorbeeld van het gebruik van binaire code in een computer is wanneer iemand typt op zijn toetsenbord. Het apparaat registreert dan die input en zendt een signaal uit over welke toets is ingetikt, in binaire code natuurlijk.

Binair talstelsel

Getallen maken met alleen nullen en enen is natuurlijk best beperkt. Daarom heeft het binair stelsel een andere manier van tellen dan hoe mensen dat gewoonlijk doen. Het binair talstelsel werkt als volgt: in een byte representeert elke bit een andere macht van 2, oplopend van rechts naar links. (Een byte van 6 bits zou dan 32, 16, 8, 4, 2 en 1 representeren.) Als één zo’n macht meegeteld moet worden, geeft de desbetreffende bit een waarde van 1. Als hij niet meegeteld moet worden, heeft de bit een waarde van 0. Neem nu als voorbeeld een byte van 6 bits: 010011. Je kunt nu met de eerder gegeven reeks uitrekenen welke waarde deze byte uitdrukt. Dat bereken je zo: 0×32 + 1×16 + 0×8 + 0×4 + 1×2 + 1×1 = 19. 010011 in het binaire talstelsel is dus gelijk aan 19 in ons talstelsel.
